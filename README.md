# Mini-Project-Data-Analyst 1

## Goals
Analisis data ini bertujuan untuk memahami kesadaran masyarakat terhadap olahraga dan dampak beberapa aktivitas olahraga dengan kalori yang terbakar. Dengan mengetahui hubungan antar variabel, kita dapat menentukan aktivitas olahraga mana yang membakar kalori paling banyak. 

## Dataset
Data yang digunakan yaitu Lifestyle Data yang berasal dari Kaggle (https://www.kaggle.com/datasets/jockeroika/life-style-data). Dataset berisi data dari sekumpumlan responden yang melakukan aktivitas fisik dengan kategori tertentu dan mencatat pola hidup dari masing-masing responden mengenai protein, karbo, dan kalori harian, durasi workout, BMI, dan sebagainya. Isi dataset dapat dilihat langsung pada website Kaggle yang sudah tertera di link.
