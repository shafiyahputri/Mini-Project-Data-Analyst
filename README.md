# Mini-Project-Data-Analyst 1

##Goals
Analisis data ini bertujuan untuk memahami kesadaran masyarakat terhadap olahraga dan dampak beberapa aktivitas olahraga dengan kalori yang terbakar. Dengan mengetahui hubungan antar variabel, kita dapat menentukan aktivitas olahraga mana yang membakar kalori paling banyak.
